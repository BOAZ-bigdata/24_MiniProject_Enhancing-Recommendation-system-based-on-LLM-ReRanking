## Project Summary
- 본 Project는 기존 딥러닝 기반 추천시스템에서 나아가, 일부 sample을 대상으로 LLM을 활용하여 추천시스템의 결과를 Re-Ranking하는 과정을 통해, 추천시스템에서의 LLM의 활용가능성을 탐구하고, 결과를 제언하고자함.
